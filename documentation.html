MyFlix API Endpoints

POST

/users

This endpoint adds a user.

Sample request

{
    "username": "jsmith",
    "password": "Test123!",
    "email": "jsmith@example.com",
    "birthday": "01/01/1980"
}

Sample response

{
    "username": "jsmith",
    "password": "Test123!",
    "email": "jsmith@example.com",
    "birthday": "1980-01-01T05:00:00.000Z",
    "favoriteMovies": [],
    "_id": "681808bcb83c1805942467d6",
    "__v": 0
}

/users/:username/favoriteMovies/:movieId

This endpoint adds a movie to a user's favorite movies list based on their username and movie ID.

Sample response

{
    "_id": "6811528de808c2cfed50e162",
    "username": "jsmith-updated",
    "password": "Test456!",
    "email": "jsmith-updated@example.com",
    "birthday": "1980-01-01T00:00:00.000Z"
}

UPDATE

/users/:username

This endpoint updates a user's information based on their username and the information provided in the request body.

Sample request

{
    "username": "jsmith-updated",
    "password": "Test456!",
    "email": "jsmith-updated@example.com",
    "birthday": "1980-01-01"
}

Sample response

{
    "_id": "6811528de808c2cfed50e162",
    "username": "jsmith-updated",
    "password": "Test456!",
    "email": "jsmith-updated@example.com",
    "birthday": "1980-01-01T00:00:00.000Z"
}

DELETE

/users/:username

This endpoint deletes a user based on their username.

Sample response

jsmith was deleted

/users/:username/favoriteMovies/:movieId

This endpoint deletes a movies from a user's favorite movies list based on their username and movie ID. The response includes an updated list of the user's favorite movies.

Sample response

{
    "message": "Movie 6811842bef7f92677a2935d4 removed",
    "user": {
      "_id": "681808bcb83c1805942467d6",
      "username": "jsmith",
      "password": "Test123!",
      "email": "jsmith@example.com",
      "birthday": "1980-01-01T05:00:00.000Z",
      "favoriteMovies": [
          "68118347ef7f92677a2935d1",
          "68118388ef7f92677a2935d2"
      ],
      "__v": 0
    }
}
