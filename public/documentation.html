MyFlix API Endpoints

CREATE

/users

This endpoint adds a new user by adding a name.

/users/:id/favoriteMovies

This endpoint adds a new movie based on the user's ID and movie name.

READ

/movies

This endpoint returns the list contained in the movies array.

/movies/:title

This endpoint returns information about the selected movie based on its title.

/movies/directors/:name

This endpoint returns information about a director based on their name.

/users

This endpoint returns the list contained in the users array.

/users/:name

This endpoint returns information about the selected user based on their name.

/genres

This endpoint returns the list contained in the genres array.

/genre/:name

This endpoint returns information about the selected genre based on its name.

UPDATE

/users/:id/name

This endpoint updates a user's name based on their ID.

/users/:id/isActive

This endpoint updates a user's status based on their ID. I added this endpoint so an inactive user can be archived instead of deleted.

DELETE

/users/:id/favoriteMovies

This endpoint deletes a movie from a user's list based on their ID.

/users/:id

This endpoint deletes a user based on their ID.
